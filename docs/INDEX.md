## 🆕 最新の機能更新

### v1.1.0 - 2024年実装

#### データベース連携機能
- PostgreSQLからリアルタイムでデータを取得
- 模擬データから実データへの移行完了
- 10種類のビールと8店舗のレストランデータを利用可能

#### ビール追加機能
- 新しいビールをデータベースに追加可能
- リアルタイムプレビュー機能
- 画像URL対応
- バリデーション機能

#### 検索機能強化
- **基本検索**: ビール名、醸造所名、説明での検索
- **カテゴリフィルター**: 6種類のビールカテゴリで絞り込み
- **アルコール度数フィルター**: 最小・最大値での範囲指定
- **醸造所フィルター**: 醸造所名での絞り込み
- **詳細フィルター**: 展開・折りたたみ機能

#### API実装
- RESTful API の完全実装
- Next.js API Routes を使用
- Prisma ORM によるデータベース操作
- 適切なエラーハンドリング
